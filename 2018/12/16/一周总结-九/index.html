发表于 2018-12-16

前言

本次总结为计算机网络中的网络层的相关概念与习题

Datagram subnet & Virtual-circuit(VC) subnet

  1. 数据报子网和虚电路子网的区别

    | Issue| Datagram subnet | Virtual-circuit subnet |
    | – | – | – |
    | Circuit setup | Not needed | Required |
    | Addressing | Each packet contains the full source and destination address | Each packet contains a short VC number |
    | State information | Routers do not hold state information about connections | Each VC requires router table space per connection |
    | Routing | Each packet is routed independently | Route chosen when VC is set up; all packets follow it |
    | Effect of router failures | None,except for packets lost during the crash | All VCs that passed through the failed router are terminated |
    | Quality of Service | Difficult | Easy if enough resources can be allocated in adance for each VC |
    | Congestion control | Difficult | Easy if enough resources can be allocated in adance for each VC |

  2. 数据报子网面向的是无连接(connectionless)，虚电路子网面向的是有连接(connection-Oriented)的
  3. 数据报子网的每个包都有完整目的地址，而路由器只需要查自己的路由表把包放到对应的端口发送出去即可，不需要修改包，因此不同包可能会有不同的路径
  4. 虚电路子网中同一个端到端系统遵循同一条路径(在传送数据之前先建立的一条特定的端到端线路)，路由器根据包头的虚电路号进行转发
  5. 虚电路子网中在转发时要修改自己的电路号

路由算法

  1. The Optimality Principle(最优化原则)

    • 如果一个路由器J在路由器I到路由器K的最优化路径上，那么从J到K的最优路径也必定遵循同样的路由
    • 汇集树(sink tree)：所有源到指定目标的路径所构成的以目标节点为根的树

  2. Shortest Path Routing(最短路径算法)

    • Dijkstra’s algorithm
  3. Flooding(泛洪算法)
    • 将每一个入境数据包发送到除了该数据包到达的那条线路以外的每条出境线路
  4. Distance Vector Routing
  5. Link State Routing
  6. Hierarchical Routing
  7. Broadcast Routing
  8. Multicast Routing
  9. Routing for Mobile Hosts
  10. Routing in Ad Hoc Networks
